Get Started with Spreadsheet Applications: Excel

Project Info:

WHAT YOU WILL LEARN
Spreadsheet applications are an essential part of productivity at work and school. Understanding how to use spreadsheet apps can make you more a more efficient and valuable worker. In this module, you will learn the essentials of working with spreadsheet apps. You will learn how to open, save, share, and print spreadsheet files, move around a worksheet and enter and edit data. You will also learn the fundamentals of using formulas and functions in spreadsheet applications. Learning Objectives Describe the basics of spreadsheet applications Describe the fundamentals of working with spreadsheet files Explain how to navigate around a worksheet and enter data Explain the basics of using formulas and functions Create and share spreadsheets, and create basic formulas
